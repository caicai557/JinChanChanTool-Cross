# 第1章 概述

- **软件版本**：JinChanChanTool v7.0.0 及以上
- **更新日期**：2026年1月22日

> **注意**：如果您使用的是旧版本软件，部分功能和界面可能与本手册描述不一致，建议更新到最新版本。

## 介绍

JinChanChanTool（后续简写为JCCT）是一款专为《云顶之弈》和模拟器上运行的《金铲铲之战》的玩家设计的辅助工具，能够帮助您在游戏对局中自动识别和拿取目标英雄卡牌，让您的注意力从牌库转移到阵容构筑和经济运营上。

JCCT基于图像识别技术，通过截取游戏画面并使用OCR（光学字符识别）技术识别商店中的英雄卡牌，然后根据您预设的目标英雄自动完成拿牌操作。

**核心工作原理**：

```
游戏画面 → 截图 → OCR识别 → 匹配目标英雄 → 模拟操作 → 完成拿牌
```

### 能帮您做什么

JCCT可以帮助您：

✅ **自动拿牌**
- 自动识别商店中的英雄卡牌
- 根据您选择的目标英雄自动拿取
- 支持鼠标模拟和按键模拟两种方式

✅ **自动刷新商店**
- 当商店没有目标英雄时自动刷新
- 配合自动拿牌实现全自动D牌

✅ **阵容管理**
- 保存和管理多套阵容
- 每套阵容支持3个子阵容变体
- 一键切换和应用大数据网站的推荐阵容

✅ **批量选择英雄**
- 按职业批量选择
- 按特质批量选择
- 快速组建阵容

✅ **装备推荐**
- 查看每个英雄的推荐装备
- 数据来源于大数据网站

✅ **多场景适配**
- 支持多显示器
- 支持任意DPI缩放
- 支持模拟器和云顶之奕

### 不能做什么（使用限制）

为了让您有合理的期望，以下是JCCT**无法做到**的事情：

❌ **不能自动决策**
- 不会自动判断应该拿哪些英雄
- 不会自动调整阵容策略
- 需要您手动选择目标英雄

❌ **不能自动对局**
- 不会自动放置英雄到棋盘
- 不会自动合成装备
- 不会自动卖出英雄

❌ **不能保证100%识别准确**
- OCR识别可能受画质、字体影响
- 少数情况下可能识别错误
- 提供纠正机制来改善识别

❌ **不支持移动端**
- 仅支持Windows系统
- 不支持iOS/Android设备
- 模拟器可以使用

> **重要提示**：JCCT是辅助工具，旨在减少重复操作，但游戏的核心决策和策略仍需要您自己完成。

### 使用场景

JCCT特别适合以下场景：

**场景1：追三星英雄**

- 快速搜索特定英雄
- 长按快捷键持续D牌
- 提高追三效率

**场景2：固定阵容上分**
- 保存常用阵容
- 一键应用阵容
- 自动拿取核心卡

**场景3：多阵容灵活转换**
- 保存多套备选阵容
- 根据场上情况快速切换
- 子阵容支持小幅调整

### 支持的赛季

程序内置多个赛季英雄池，可随时切换：

- ✅ S16 英雄联盟传奇（当前主赛季）
- ✅ 强音对决
- ✅ 其他赛季...
